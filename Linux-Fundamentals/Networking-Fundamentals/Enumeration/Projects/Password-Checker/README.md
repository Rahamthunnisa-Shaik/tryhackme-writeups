# Password Strength Checker Project

## Overview
A Python script that checks passwords for strength based on length, character types, and common patterns.

## Tools & Languages Used
- Python 3
- Regular Expressions

## Features
- Checks for minimum length (8 characters)  
- Ensures use of uppercase, lowercase, numbers, and symbols  
- Warns about common passwords
