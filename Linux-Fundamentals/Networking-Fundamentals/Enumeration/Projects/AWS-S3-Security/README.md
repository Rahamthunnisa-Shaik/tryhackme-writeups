# AWS S3 Bucket Security Project

## Overview
This project demonstrates securing AWS S3 buckets using IAM roles and bucket policies.

## Tools Used
- AWS Management Console
- IAM Roles and Policies
- S3 Bucket Configuration

## Key Learnings
- Implemented least privilege access  
- Configured bucket policies to prevent public access  
- Tested access with different IAM roles  
- Learned AWS best practices for cloud security
