# Linux Hardening Project

## Overview
This project demonstrates basic Linux system hardening to improve security posture.

## Tools & Commands Used
- chmod / chown
- sudo management
- firewall configuration (ufw)
- file permission audits

## Key Learnings
- Secured file permissions and ownership  
- Created limited privilege users  
- Configured basic firewall rules  
- Learned steps to reduce attack surface on Linux systems
